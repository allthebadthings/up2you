### Web Applications (HTML + JavaScript)

#### 1. **Jewelry_Image_Uploader.html**
**Tech Stack:**
- Pure HTML/CSS/JavaScript
- Imgur API (anonymous upload)
- Client-side only, no backend

**Purpose:** Bulk upload jewelry images to Imgur, get .jpg URLs for inventory

**API Used:**
- Imgur API v3
- Client ID: `546c25a59c58ad7` (anonymous uploads)
- Endpoint: `https://api.imgur.com/3/image`
- No authentication required
- Rate limit: ~1250 uploads/day per IP

**How it works:**
1. User drops image files
2. JavaScript uploads via FormData to Imgur API
3. Returns direct image URL (https://i.imgur.com/xxxxx.jpg)
4. Auto-copies URL to clipboard
5. User pastes in Excel

**No setup required** - works immediately

#### 3. **Bundle_Builder.html**
**Tech Stack:**
- HTML5 Canvas API
- SheetJS (XLSX.js) for Excel parsing
- Drag-and-drop API
- Pure JavaScript

**Purpose:** Visual bundle creation with auto-generated marketing images

**How it works:**
1. User uploads Excel inventory file
2. SheetJS parses the "Finished Inventory" sheet
3. Displays items in searchable left panel
4. User drags items to canvas or enters SKUs
5. Calculates pricing with discounts
6. Generates composite image on canvas showing all items
7. Downloads bundle image OR uploads to Imgur

**Dependencies:**
- SheetJS CDN: `https://cdnjs.cloudflare.com/ajax/libs/xlsx/0.18.5/xlsx.full.min.js`
- Imgur API (same as #1)

**Image Generation:**
- Canvas draws 1200x(dynamic height) image
- Header with bundle name
- Grid layout of product images (3 per row)
- Each item shows: image, SKU, name, price
- Footer with total price and item count
- Professional gradient background

**Cross-Origin Note:**
Images must be CORS-enabled or from same origin. Shopify CDN images work. Some sources may fail due to CORS.

## ğŸ› TROUBLESHOOTING

### Common Issues

**1. "Images not uploading to Imgur"**
- Check internet connection
- Imgur has rate limits (1250/day per IP)
- File must be image format (jpg, png, gif)
- Max file size: 10MB

**2. "Bundle Builder won't load Excel file"**
- File must be .xlsx format (not .xls or .csv)
- Must have sheet named "Finished Inventory"
- Check browser console for errors (F12)

**3. "Excel formulas showing errors"**
- Don't delete columns (breaks VLOOKUP references)
- Ensure LibreOffice/Excel is up to date
- Check regional settings (comma vs semicolon in formulas)

**4. "Product photos look weird"**
- Images with complex backgrounds work less well
- Try photos on plain/neutral backgrounds
- Canvas compositing works best with centered jewelry

**5. "Cross-origin image errors in Bundle Builder"**
- Some image hosts block cross-origin access
- Shopify CDN works fine
- Imgur works fine
- Private servers may need CORS headers
